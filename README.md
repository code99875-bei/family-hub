# Family Hub — 家庭管理中心

整合家庭行事曆、家計簿與個人財務的多人共用管理系統。

## 功能規劃

- 家庭行事曆（多人共用）
- 家計簿（收支記帳、月報）
- 個人財務分區（權限分離）

## 技術架構

| 層級 | 技術 |
|------|------|
| 前端 | HTML / CSS / JavaScript（日系溫馨繪本風） |
| 資料庫 | Supabase |
| 部署 | Vercel |
| 登入 | Supabase Auth（Google 帳號）|

## 資料夾結構

```
family-hub/
├── src/
│   ├── css/          # 樣式
│   ├── js/           # 邏輯
│   └── pages/        # 各功能頁面
├── assets/           # 圖片、字型等靜態資源
└── docs/             # 規格文件
```

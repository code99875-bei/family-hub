# Family Hub 功能需求文件

## 專案概述
- 名稱：Family Hub 家庭管理中心
- 架構：HTML/CSS/JavaScript + Supabase + Vercel
- 登入：Supabase Auth（Google 帳號）
- 使用者：貝（我）、群（先生）

| 代稱 | 身份 | 顏色 | 色碼 |
|------|------|------|------|
| 貝 | 我 | 天藍色 | #5BB8E8 |
| 群 | 先生 | 深紫色 | #7B5EA7 |
| AZ | 兒子 | 深藍色 | #1A3A6B |
| Emma | 女兒 | 玫瑰粉 | #F4A7B9 |
| 爺爺 | 公公 | 深綠色 | #2D6A2D |

## 視覺風格
- 日系溫馨繪本風，走可愛路線
- 操作介面：直覺如 POS 機，按鈕大、點擊流暢
- 字體：圓體為主（Noto Sans TC）
- 配色以各成員代表色為主軸貫穿全站

## 頁面架構
1. 首頁儀表板
2. 家庭行事曆
3. 家計簿
4. 個人財務（登入後各自獨立）

## 一、首頁儀表板
- 本週＋下週事件摘要（成員顏色點點）
- 本月家計簿結餘
- 個人財務本月餘額（僅顯示本人）
- 當月先生未撥款且已到月底 → 顯示灰字「本月未撥款」

## 二、家庭行事曆

### 事件欄位
- 事件標題（必填）
- 所屬成員（可複選，必填）
- 日期（必填）
- 開始時間～結束時間（必填）
- 預計出門時間（選填）
- 預計回家時間（選填）
- 備註（選填）

### 重複事件
- 每天／每週（選星期幾）／每月（選幾號）
- 重複區間：起始日期＋結束日期
- 可單獨修改某一次，不影響其他

### 衝突警示
- 同一天貝與群都有事件 → 跳窗：「⚠️ 這天你們都有行程，請確認爺爺阿嬤是否可以幫忙！」

### 月總覽
- 彩色點點（每成員一個顏色）
- 點擊日期展開行程
- 截圖模式：隱藏操作介面

## 三、家計簿

### 收入
- 每月1號自動產生「貝的固定撥款 3,000元」
- 先生撥款：手動輸入（日期＋金額＋備註）
- 若當月最後一天先生無撥款 → 灰字「本月未撥款」

### 支出欄位
- 日期、金額、類別、付款方式、備註、登記人（自動帶入）
- 類別：飲食／交通／醫療／教育／日用品／娛樂／孝親費／其他
- 付款方式：現金／轉帳／刷卡

### 統計
- 本月結餘（收入－支出）
- 各類別支出圖表
- 可切換月份

## 四、個人財務

### 貝的帳戶清單
| 帳戶 | 類型 |
|------|------|
| 郵局 | 存款帳戶 |
| 富邦 | 存款帳戶 |
| 中國信託 | 存款帳戶 |
| LINE Bank | 存款帳戶（預留）|
| 手邊現金 | 現金 |
| 緊急預備金 | 現金（重大事件才動用）|

### 付款方式與帳戶對應
- 現金：手邊現金／緊急預備金
- 轉帳：郵局／富邦／中信／LINE Bank
- 信用卡刷卡：郵局／富邦／中信／LINE Bank

### 收支紀錄
- 日期、金額、收入／支出、付款方式、使用帳戶、備註
- 緊急預備金動用時 → 特別標色提示

### 信用卡對帳
- 篩選刷卡支出，按月份匯總
- 顯示本月刷卡總額＋各筆明細
